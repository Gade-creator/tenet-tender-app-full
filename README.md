# TENET Tender Ecosystem — Full Application

## 📱 Overview
This is the full implementation of the TENET Tender Ecosystem mobile application.

The platform connects:
- Tender Owners (Companies)
- Contractors / Suppliers
- Administrators

---

## 🚀 Features

### 🔐 Authentication
- User registration & login
- Role-based access

### 📊 Dashboard
- Personalized user dashboard
- Tender recommendations

### 📄 Tender Management
- Post tenders (for companies)
- View tenders
- Filter & search tenders

### 💰 Bidding System
- Submit bids
- View submitted bids
- Bid tracking

### 💬 Messaging
- Real-time communication between users

### 👤 Profile
- Profile management
- Document upload
